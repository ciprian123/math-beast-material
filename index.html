Adunarea numerelor

Adunarea este o operație aritmetică elementară care totalizează două sau mai multe numere, numite „termenii adunării” într-o singură valoare, numită suma sau „totalul” numerelor respective. Aceasta se poate realiza prin numarare, sisteme numerice, calculator.

1 + 2 + 3 + ... n = n * (n + 1) / 2;
1234 + 4567 = 6912
2 + 2 + 2 + 2 = 2 * 4 = 8
a + b = b + a
0 + a = a + 0 = a
1 + 1 = 2

Scaderea numerelor

Scăderea este una dintre cele 4 operații aritmetice elementare; este inversul adunării, însemnând că dacă începem cu orice număr la care adăunăm orice numar, apoi scădem numărul pe care l-am adunat, ne întoarcem la numărul cu care am început. Scăderea este reprezentată prin semnul minus.

12345 - 12345 = 0
1234 + (-) 1234 = 0
8 - 2 - 2 - 2 - 2 = 0
a + b = b + a
a - 0 = a
2 - 1 = 1

Inmultirea numerelor

Inmulțirea este o operație matematică ce poate fi definită ca o adunare succesivă.
Ex: 3 × 4 = 3 + 3 + 3 + 3 = 12.
Inmultirea este o operatie comutativa:
Ex: a × b = b × a
Inmultirea este o operatie asociativa:
(a × b) × c = a × (b × c).
Inmultirea este o operatie distributiva fata de adunare/scadere:
Ex: x × (z + y) = x × z + x × y ,
Ex: x × (z - y) = x × z - x × y.
Inmultirea este o operatie pentru care 1 este element neutru:
Ex: 1 × x = x.
Inmultirea este o operatie pentru care, dacă un factor este 0, atunci rezultatul va fi egal cu 0:
Ex: x×0=0.

Radicali

In matematica, radicalul unui numar reprezinta un alt numar, care ridicat la o putere (denumita ordinul radicalului) este egal cu numarul dat. Daca ordinul unui radical nu este precizat, atunci este vorba implicit de radicalul de ordinul 2, denumit si radacina patrata.
Ex: 2 * 2 = 4
Radical din 2 este 1,4142135623...
Radical din 3 este 1,7320508075...
Radical din 4 este 2

Radical din 6 este 2,4494897427...

Puteri

Puterea n a unui numar a, notată an, este o operatie dintre aceste numere, numite baza, respectiv exponent. In vorbire an se pronunta "a la puterea n", sau, pe scurt, "a la n", Dacă n este un numar natural, atunci ridicarea la putere poate fi definita ca o inmultire repetata:
Ex: 23 = 2 * 2 * 2
Ex: 0n = 1
Ex: 33 = 3 * 3 * 3

Orice numar la puterea 0 este 1!

Conversii intre baze de numeratie

Prin conversia unui numar dintr-o baza de numeratie in alta realizam o serie de operatii matematice elementare. Atunci cand trecem un numar din baza de numeratie a in baza de numeratie b impartim numarul a la b atata timp cat acesta este mai mare ca zero iar resturile obtinute in urma impartirii numarului a la b scrise in ordine inversa reprezinta numarul a scris in baza b.
Ex:

12(10) = 1100(2)
123(10) = 11110110(2)
188(10) = BC(16)
