Resurse utilizate

HTML 5

HTML 5 este ultima versiune a limbajului HTML, cea mai utilizata tehnologie in scrierea site-urilor web. Limbajul HTML este unul simplu si usor de invatat, acesta ruleaza in majoritatea browserelor moderne (Google Chrome, Mozilla Firefox, Microsoft Edge, Opera). Acest limbaj poate fi scris in majoritatea editoarelor de texte (Notepad, Text Mate, Gedit, Atom, Brackets) si nu necesita alte programe spaciale.

CSS 3 (SCSS)

CSS (Cascading Style Sheet) este un limbaj utilizat in dezvoltarea web pentru a "controla" partea de design a unei pagini web. Acesta este un standard de la aparitia limbajului HTML 5 deoarece limbajul HTML nu a fost nicioata gandit pentru design, ci pentru a defini structura paginii. Pentru a intruduce CSS in pagina web se utilizeaza tagul < style > care poate fi inserat fie in < head > , fie inline, fie in fisiere externe cu extenisia .css, incluse prin tagul < link >. JavaScript (jQuery)

JavaScript reprezinta unul dintre cele mai populare limbaje de programare, fiind deosebit de important dezvoltarea web, atat in partea de frontend cat si in partea de backend, aceasta fiind construita pentru a rula in browser in mod special. Desi JavaScript beneficiaza de numeroase framework-uri, jQuery nu este un fremwork, ci o librarie de cod JavaScript care simplifica deosebit de mult munca programatorului sau a designer-ului.
