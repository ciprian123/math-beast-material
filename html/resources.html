Resurse utilizate

HTML 5

HTML 5 este ultima versiune a limbajului HTML, cea mai utilizata tehnologie in scrierea site-urilor web. Limbajul HTML este unul simplu si usor de invatat, acesta ruleaza in majoritatea browserelor moderne (Google Chrome, Mozilla Firefox, Microsoft Edge, Opera). Acest limbaj poate fi scris in majoritatea editoarelor de texte (Notepad, Text Mate, Gedit, Atom, Brackets) si nu necesita alte programe spaciale.
Pentru mai mult cititi documentatia oficiala

CSS 3 (SCSS)

CSS (Cascading Style Sheet) este un limbaj utilizat in dezvoltarea web pentru a "controla" partea de design a unei pagini web. Acesta este un standard de la aparitia limbajului HTML 5 deoarece limbajul HTML nu a fost nicioata gandit pentru design, ci pentru a defini structura paginii. Pentru a intruduce CSS in pagina web se utilizeaza tagul < style > care poate fi inserat fie in < head > , fie inline, fie in fisiere externe cu extenisia .css, incluse prin tagul < link >. Pentru mai mult cititi documentatia oficiala

JavaScript (jQuery)

JavaScript reprezinta unul dintre cele mai populare limbaje de programare, fiind deosebit de important dezvoltarea web, atat in partea de frontend cat si in partea de backend, aceasta fiind construita pentru a rula in browser in mod special. Desi JavaScript beneficiaza de numeroase framework-uri, jQuery nu este un fremwork, ci o librarie de cod JavaScript care simplifica deosebit de mult munca programatorului sau a designer-ului.
Documentatie Javascript Documentatie jQuery

Jetbrains WebStorm

WebStorm este un IDE creat de catre dezvoltatorii de la JetBrains, aceeiasi care au creat si IntelliJ idea, binecunoscutul Java IDE. WebStorm este folosit in special pentru web development, acesta avand suport pentru o varietate foarte mare de tehnologii web, este customizabil si beneficiaza de un numar impresionant de plugin-uri create atat de catre Jetbrains cat si de catre comunitate.

Google Chrome

Pe parcursul realizarii acestui proiect, acesta a fost testat cu succes pe navigatorul Google Chrome. Google Chrome este un navigator web de tip open source dezvoltat de compania Google. Este cel mai utilizat browser la momentul actual, având o cotă de piață de 52.1 % din totalul utilizatorilor de internet. Numele navigatorului este derivat de la interfața grafică. Paperplane.io (hosting)

Datorita faptului ca acest proiect nu utilizeaza tehnologii server-size (PHP, Ruby on Rails, Python), nu este necesar ca acesta sa fie hostat pe site-urile clasice de webhosting a caror specificatii variaza foarte mult (pret, latime de banda, domenii). Utilizand hosting-ul oferit de cei de la paperplane.io utilizatorul are posibilitatea sa stocheze datale site-ului chiar in contul sau de dropbox, sau github, in mod gratuit.
